Diseño e Implementación de una Función en C++

En esta página, explicaremos el proceso completo de diseño e implementación de una función en C++. Como ejemplo, implementaremos una función que calcula el factorial de un número.

Introducción

El factorial de un número n (denotado como n!) es el producto de todos los enteros positivos desde 1 hasta n. Por ejemplo, 5! = 5 × 4 × 3 × 2 × 1 = 120.

Diseño de la Función

Antes de implementar la función, es importante definir su propósito, parámetros, tipo de retorno y posibles casos especiales.

  • Propósito: Calcular el factorial de un número entero no negativo.
  • Parámetros: Un entero n.
  • Tipo de Retorno: Un entero que representa el factorial de n.
  • Casos Especiales: El factorial de 0 es 1 por definición.

Implementación en C++

A continuación se muestra el código de la función en C++:

#include <iostream>
using namespace std;

int factorial(int n) {
    if (n < 0) {
        cout << "Error: El número debe ser no negativo." << endl;
        return -1;
    }
    if (n == 0) {
        return 1;
    }
    int result = 1;
    for (int i = 1; i <= n; ++i) {
        result *= i;
    }
    return result;
}

int main() {
    int number;
    cout << "Introduce un número: ";
    cin << number;
    int fact = factorial(number);
    if (fact != -1) {
        cout << "El factorial de " << number << " es " << fact << endl;
    }
    return 0;
}

Este código define una función factorial que calcula el factorial de un número entero no negativo.

Conclusión

Hemos diseñado e implementado una función en C++ que calcula el factorial de un número. Este ejemplo ilustra cómo definir el propósito de la función, manejar casos especiales y estructurar el código en C++.
